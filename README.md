# AI Document Intelligence (ADi)

A premium, world-class document analysis assistant built with **Streamlit** and **LiteLLM**. ADi extracts structured insights from PDFs, DOCX, and XLSX files using various LLM providers.

## 🚀 Features
- **Multi-Provider Support**: Choose between Google Gemini, OpenAI, Anthropic, and Perplexity.
- **Premium Design**: Modern "Slate & Indigo" theme with glassmorphism and smooth animations.
- **Unified Logic**: Simplified LLM interactions for consistent extraction.
- **Knowledge Base**: Instantly scan and query multiple uploaded documents.

## 🛠️ Installation
1. Clone the repository:
```bash
git clone https://github.com/JoushaNandhu/Document.git
cd Document
```
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Run the app:
```bash
streamlit run app.py
```

## 🌐 Deployment
This application is ready for deployment on **Streamlit Cloud**:
1. Connect your GitHub account to Streamlit.
2. Select this repository and the `main` branch.
3. Set `app.py` as the main entry point.
4. Add your API keys as secrets/environment variables if needed.
